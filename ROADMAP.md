# IRE Roadmap

## Polish the Core Experience
- Finish onboarding (first-launch tutorial)
- Drag-and-drop context, richer tool cards, smoother chat/composer UX

## Deepen Context & Memory
- Automated wiki linting — catch contradictions, orphaned notes, stale claims
- Better cross-linking between ideas, resources, and experiments

## Agentic Workflows
- `/ire-init` — bootstrap a workspace from questions + context scan
- `/ire-lint` — the wiki-health check above, as a workflow
- Background multi-experiment agent with autonomous dispatch

## Expand Reach
- Broader model/provider support with auto-discovery
- Multi-machine support — local, HPC, remote workstations
- Agent multiplexing across sessions via git worktrees
