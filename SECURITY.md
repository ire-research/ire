# Security Policy

## Supported Versions

IRE is pre-1.0 and under active development. Security fixes are made
against `main` only — there are no maintained release branches yet.

## Reporting a Vulnerability

Please **do not open a public GitHub issue** for security vulnerabilities.

Instead, report it privately by emailing both maintainers:

- giacomociro02@gmail.com (Giacomo Cirò)
- rossi.vittori@gmail.com (Vittorio Rossi)

Include as much detail as you can: the affected component, steps to
reproduce, and the potential impact. We'll acknowledge your report within
**3 business days** and aim to provide a fix or mitigation plan within
**30 days**, depending on severity and complexity.

If the issue is confirmed, we'll credit you in the fix's release notes
unless you prefer to remain anonymous.
